Internet

The internet is a gloabl network of interconnected computers and devices that allows people to share information, communicate, and acess services.

Web

It refers to World Wide Web (WWW), where users can access information using browisng applications.

Difference between Internet and Web

Web is a component of the internet, but not internet itself. The internet can communicate with other computers but a web can't but it can be accessed from any other computer.

ASCII and Unicode

ASCII stands for American Standard Code for Information Interchange. Basically what it does is it assigns a number for each character which includes letters, digits, punctuation and special characters. Unicode is more complicated encoding system that can assign or support thousands of characters from different languages.

UTF 8

It stands for Unicode Transformation Format which is 8 bit. It can support or assign different style and a lot of characters from different languages ands styles while ASCII can only take 0-127 characters.

URL

It stands for Uniform Resource Locator which is a web adress that you can enter in a browser to get to a specific location.

Domain Name

It is a name that is part of the URL. It is easily readable for humans and tell what the location is about.

HTTP

HTTP stands for Hypertext Transfer Protocol which is used for transmitting datat on the web.

HTTPS

It stands for Hypertext transfer protocol secure which is a secure version of HTTP. This is more trustworthy and encrypted.

IP Adress

Internet Protocol address is a unique number that is given to every device on the network. Witht the IP address one can tell it's location.

DNS

Domain Name System works as a translator from human-readable domain names into system readable IP addresses. So, the users can access a website not by using the ip address number but by simply with a human readable name.

W3C Recommendations

This is a guidelines and standards developed by W3C to better align and maintain the web. Anyone who design a web page should follow guidelines.

XML, XHTML, HTML, HTML5

Extensible Markup Language (XML) is used to store and transport data. Extensible Hypertext Markup Language (XHTML) ensures that web documents are well-formed. Hpertext Markup Language (HTML) is a language for creating web pages. HTML5 is the latest version of HTML, which is upgraded to new features like audio and video.
