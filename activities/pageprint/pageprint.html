1. The viewport meta tag helps a web page fit better on different screen sizes, especially on mobile devices. It tells the browser how to adjust the layout for smaller screens so users can read and navigate easily.

2. When you use the same CSS property twice, the last one written takes priority. So, if flex-direction: column; is written after flex-direction: row;, the layout will use the column direction.

Use media queries to create styles for smaller screens, like phones, making layouts and text easier to see. For example, "media" allows you to apply specific CSS when the screen is narrower than 600 pixels.
