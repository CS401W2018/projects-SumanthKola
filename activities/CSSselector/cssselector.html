List out the different ways to select an item.

You can select an item using ID or Class and assigning a specific name for them.

What is the difference between an ID and a Class?

An ID is a selector that can't be re-used and unique name usually as nouns. A Class is also a selector but can be re-used usually as adjectives.

What is the difference between a SPAN tag and a DIV tag?

It is used to select a certain part of the code and change its style and it also doesn't cause line break. DIV tag covers multiple tags like a invisible box.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

header and footer usually refer to the header of the page and the footer of the page, respectively. However div can refer to any general section of a page.

What are different ways to select a color?

You can select just by typing the name of the color you desire or even select the shades of it from the color grid.
