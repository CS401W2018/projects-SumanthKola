Explain the difference between inline, embedded, and external CSS.

Inline css goes in main tag withing p tags along with the paragraphs. Embedded css goes in head tag at the beginnin of the code separate from the paragrapgs. External css goes in a different .css file outside of the original file which is linked to the file using link:css

Explain why you should only use external CSS.

This is for the graphic designer to develop the page into styles by not touching the html coding.
